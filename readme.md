# MarketPulse AI

MarketPulse AI is a Multi-Agent Equity Research Platform that automates stock analysis using AI agents, financial data, market news, and investment decision-making workflows.

The platform enables users to enter a company name or stock ticker and receive a comprehensive investment analysis including research insights, financial evaluation, market sentiment, risk assessment, bullish and bearish viewpoints, and a final investment recommendation.

---

## Features

### Multi-Agent Analysis

The system uses multiple specialized AI agents:

- Research Agent
- Financial Agent
- News Agent
- Bull Agent
- Bear Agent
- Risk Agent
- Investment Committee Agent

Each agent performs a specific task and contributes to the final investment recommendation.

### Stock Intelligence

- Company profile generation
- Market snapshot
- Financial analysis
- News sentiment analysis
- Bull case generation
- Bear case generation
- Risk assessment
- Buy / Hold / Sell recommendation

### Dashboard

- Interactive Streamlit interface
- Company information
- Stock performance charts
- Risk score visualization
- Confidence score display
- Watchlist management
- PDF report generation

### Smart Search

Users can search using:

- Company names
- Stock tickers
- Major US stocks
- Major Indian stocks

Examples:

- NVIDIA
- Apple
- Tesla
- Reliance
- Infosys
- HDFC Bank
- NVDA
- AAPL

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/MarketPulse-AI.git

cd MarketPulse-AI
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

Mac/Linux:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Environment Variables

Create a `.env` file:

```env
GROQ_API_KEY=your_groq_api_key
```

## Running Locally

### Backend

```bash
uvicorn app.main:app --reload
```

Backend runs at:

```text
http://127.0.0.1:8000
```

### Frontend

```bash
streamlit run frontend/streamlit_app.py
```

Frontend runs at:

```text
http://localhost:8501
```
